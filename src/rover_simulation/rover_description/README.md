# `robot_description` ROS2 Package

The `robot_description` ROS2 package is designed to provide comprehensive definitions for links, joints, sensors, plugins, and configuration files crucial for the rover model. Additionally, a dedicated launch file has been included to streamline the visualization of the robot model in RViz2.

## Directories:
1. **urdf:** Contains all Xacro files for the model description.
2. **config:** Configuration files for velocity controller.
3. **models:** SDF description of the model.
4. **images:** Images of the rover in RViz and Gazebo.
